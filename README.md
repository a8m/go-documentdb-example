# Go-DocumentDB-Example

> A users CRUD app using Martini and DocumentDB

### Install

**(1)** Clone this repo
```sh
$ go get github.com/a8m/go-documentdb-example
```
**(2)**
edit config file with your `connection-url` and `masterKey`

### Run the app

```sh
$ go build && ./golang-documentdb-example
```
